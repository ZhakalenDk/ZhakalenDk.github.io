  • Update 1.4.2.0 - Bugfixes and new Launcher

    This Update is a milestone! We're finally done with our Game Loop design and are now able to begin building our game!
    For you this means that from now on, we will work with deadlines as you have been doing so far.
    The team is working on a schedule for you which you can see below.

    Size of patch Description Development time
    Major Update: These are the milestone updates. They are heavy and will always include major changes such as, new parts of the world, Aesthetic overhauls, new foes/NPC's etc. The updates are so heavy it will depend on how much time we need to get it ready. However they will be put out along with the regular smaller updates.
    Minor Update: These are the smaller Updates. They will include small bugfixes, new smaller features such as extended controls and Testers tools (Cheat systems) etc. The updates will be put out on a regular 14 days basis. Unless listed otherwise.

    On 23rd of November we will release the a detailed schedule that will cover the updates for the rest of the year

    What's new?

    As the 4th version of the Game Loop is a combination of every basic feature you have been testing so far, it will not contain any new features besides "God Mode", which is a Testers Tool to quickly travel the world while flying along with 10X health, stamina and damage.(Just a "I'm immortal" mode in case you got bored of running)
    There is one major change though!
    We're happy to announce that we've replaced Unity's default Launcher with our own hand written Launcher.
    The launcher will be updated as we progress through the development. We're working on a patch system to spare you of the trouble of downloading the whole game every single time we update the build.
    We're sorry that you have to go through that... Really... We are!

    How does it work?

    From the Launcher you can of course launch the game, but you can also tag along on our newsfeed (And visit our Website... But that's not as cool is it?).
    The launcher has access to a newsfeed server, which contains patch notes and other information. You will not have to reinstall the Launcher in order to get your news.
    The best part is.. You do not have to go to Trello (And sign in) in order to download a new build.
    The Launcher will detect if there's a new version available and prompt you to download and install it. However, as stated above the system will download the whole game everytime which is not a clean solution, we're working on that!

    We'll be giving some more information on the "Patch system" progress when we're ready to show you!
    Enjoy and happy testing!
